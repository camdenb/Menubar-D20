Menubar d20

A tiny dice set that lives in your OS X menu bar!

View on GitHub Download .app

Aren't you tired of wasting screen space with a whole window or webpage dedicated to a dice roller? With Menubar d20, a quick throw of the dice is just a click away.

Features

  • Clean, intuitive interface updated for Yosemite
  • Easily add modifiers or change the number of rolls
  • Support for all of the traditional dice: from d4 to d100
  • View a breakdown of each roll, including the minimum and maximum roll
